# Portal — Glavni Ulazni Portal

Ovo je glavni landing page za **AI-IQ Super Platformu**.

## Opis

Portal služi kao centralna tačka ulaska za sve platforme unutar monorepo-a. Korisnici mogu odavde pristupiti svim dostupnim platformama.

## Pokretanje

```bash
npm run dev
```

Otvori `http://localhost:3000` u browseru.

## Fajlovi

```
portal/
├── public/
│   ├── index.html     ← Glavna stranica
│   ├── style.css      ← Stilovi portala
│   └── main.js        ← JavaScript logika
└── src/               ← Izvorni kod (za buduće proširenje)
```
