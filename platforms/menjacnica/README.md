# Ai IQ Menjačnica 💱

**Platforma za menjanje valuta sa AI predikcijama i analizom tržišta.**

Deo [AI-IQ Super Platforme](../../README.md).

## Opis

Ai IQ Menjačnica je platforma koja koristi veštačku inteligenciju za praćenje kurseva valuta, predikciju kretanja deviznog tržišta i optimizaciju menjačkih operacija.

## Pokretanje

```bash
npm run dev
```

Otvori `http://localhost:3003` u browseru.
