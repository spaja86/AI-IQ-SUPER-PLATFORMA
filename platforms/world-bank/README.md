# Ai IQ World Bank 🏦

**Svetska banka platforma za finansijske usluge i AI analitiku.**

Deo [AI-IQ Super Platforme](../../README.md).

## Opis

Ai IQ World Bank platforma pruža finansijske usluge zasnovane na veštačkoj inteligenciji — analitiku tržišta, upravljanje portfolijima, i AI predviđanja kretanja.

## Pokretanje

```bash
npm run dev
```

Otvori `http://localhost:3002` u browseru.
