# IO OpenUI AO 🌐

**Platforma za otvoreni korisnički interfejs sa naprednim AI mogućnostima.**

Deo [AI-IQ Super Platforme](../../README.md).

## Opis

IO OpenUI AO je platforma koja pruža otvoreni (open-source) UI sistem sa podrškom za AI integracije. Namenjena je za kreiranje modernih, responzivnih i AI-podržanih korisničkih interfejsa.

## Pokretanje

```bash
npm run dev
```

Otvori `http://localhost:3001` u browseru.

## Struktura

```
io-openui-ao/
├── src/
│   ├── index.html     ← Glavna stranica
│   ├── style.css      ← Stilovi
│   └── main.js        ← Logika
└── package.json
```
