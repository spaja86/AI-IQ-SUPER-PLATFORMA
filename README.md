## 🏁 Pokretanje

```bash
npm install
npm run dev
```

Otvori [http://localhost:3000](http://localhost:3000) u pregledaču.

## 📡 API

- `GET /api/status` — Status svih platformi, sektora i IT proizvoda u JSON formatu

## ▲ Vercel Deployment

```bash
npm run build
```

Ili deploy direktno na Vercel povezivanjem GitHub repozitorijuma.
